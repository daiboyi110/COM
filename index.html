Data & Privacy Notice

Please read carefully before using this website. You must agree to continue.

1. What data is collected and by whom

  • Visit counter — a single anonymised count is incremented in a Firebase Realtime Database operated by the web designer each time a new browser session begins. No IP address, name, or any personal identifier is stored alongside this number.
  • Firebase Analytics (Google LLC) — Firebase Analytics is active on this site. Google may collect your approximate geographic region, device type, browser type, operating system, language, and session duration. Your exact IP address is not stored by the web designer; Google applies IP anonymisation where supported. Google may use this data in accordance with its own privacy policy. See Google's Privacy Policy.
  • Firebase SDK — Google CDN (gstatic.com) — the Firebase JavaScript libraries are loaded from Google's own CDN (www.gstatic.com). When your browser fetches these three script files on page load, Google's CDN servers may log your exact IP address, browser User-Agent (browser name, version, OS), referrer URL, and request timestamp. This occurs before any user interaction and is outside the web designer's control.
  • Firebase Realtime Database connection — when the page initialises, it opens a live HTTPS/WebSocket connection to Google Firebase servers (pose-b0689-default-rtdb.firebaseio.com) to update the visit counter. Google Firebase servers may log your exact IP address and connection metadata as part of this request.
  • GitHub Pages (website host) — this website is hosted on GitHub Pages (GitHub, Inc., a Microsoft subsidiary). GitHub's web servers may automatically record your exact IP address, browser User-Agent, referrer, and request timestamp in standard server access logs for every page request. This is a standard practice of all web hosts and is entirely outside the web designer's control. See GitHub's Privacy Statement.

Summary of exact IP address storage by party: Web designer — NO  |  Google Firebase Analytics — NO (anonymised)  |  Google Firebase CDN / Database — YES (server logs)  |  GitHub Pages — YES (server logs)  |  jsDelivr CDN — YES (server logs)  |  SheetJS CDN — YES (server logs)

2. Third-party libraries — CDN metadata

The following libraries run entirely within your browser and perform no analysis on any server. However, your browser must download each library file from an external CDN when the page first loads. Each CDN request exposes your exact IP address, browser type, OS, and referrer to that CDN provider:

  • MediaPipe Pose (Apache 2.0 License, Google LLC) — four library files are fetched from jsDelivr CDN (cdn.jsdelivr.net). jsDelivr may log your exact IP address and connection metadata. See jsDelivr's Privacy Policy. Once downloaded, all pose detection runs locally using your device's CPU/GPU via WebAssembly; no video frames, images, or landmark data are ever transmitted.
  • SheetJS Community Edition — one library file is fetched from SheetJS CDN (cdn.sheetjs.com). SheetJS's CDN provider may log your exact IP address and connection metadata. See SheetJS's Privacy Policy. Once downloaded, all Excel file generation runs locally; no spreadsheet data is ever transmitted.

3. Camera access

If you choose to use the live camera feature, your browser will display a permission prompt asking for access to your camera. Please note:

  • Camera access is optional — the tool also works with uploaded video files and images without any camera permission.
  • The live camera feed is processed entirely within your browser by MediaPipe Pose. No video frames or images from your camera are ever transmitted to any server.
  • The web designer has no access to your camera feed at any time.
  • You can revoke camera permission at any time through your browser settings.
  • Microphone access is never requested by this website.

4. Cookies and browser storage

  • Google Analytics cookies — Firebase Analytics may set first-party cookies (such as _ga and _gid) in your browser to distinguish unique sessions and users. These cookies are set and read by Google's analytics scripts and are subject to Google's cookie policy. You can block or delete these cookies through your browser settings.
  • sessionStorage (local only) — this website stores two small flags in your browser's sessionStorage: one to record that you have given consent, and one to avoid counting the same browser session twice in the visit counter. sessionStorage data is stored only on your device, is never transmitted to any server, and is automatically deleted when you close the browser tab.
  • No tracking cookies by the web designer — the web designer does not set any cookies, does not use any cross-site tracking technology, and does not perform browser fingerprinting.

5. What is NOT collected or uploaded

  • Your uploaded videos and images — processed locally in your browser only, never uploaded.
  • Your pose / motion / landmark data — computed locally, never transmitted.
  • Your exported Excel files — generated locally and saved directly to your device; no data is sent to any server during export.
  • Your name, email, or any account information — this website has no user accounts and no login system.
  • Microphone audio — never requested or accessed.
  • Browser fingerprint or device identifiers — not collected by the web designer.

6. Data retention

  • Visit counter (web designer / Firebase) — the anonymous count is retained indefinitely as an aggregate number with no personal data attached.
  • Google Firebase Analytics — Google retains analytics data for up to 26 months by default, subject to Google's data retention settings and privacy policy.
  • CDN access logs (Google gstatic.com, jsDelivr, SheetJS CDN) — retention periods are governed by each provider's own policy.
  • GitHub Pages server logs — governed by GitHub's privacy statement; typically retained for a limited period for security and operational purposes.
  • sessionStorage — deleted automatically when you close the browser tab.

7. Security

  • All connections between your browser and external services (GitHub Pages, Firebase, CDNs) use HTTPS encryption.
  • No user-generated content (videos, images, pose data) is stored on any server.
  • The web designer does not have access to any server logs held by GitHub, Google, jsDelivr, or SheetJS.

8. Children's privacy

This website is suitable for all ages. The web designer does not collect any personal information from any user, including children. However, third-party services used by this site (Google Firebase Analytics, GitHub Pages, and CDN providers) may log connection metadata such as IP addresses as part of standard server operations — this applies to all visitors regardless of age. If you are under 13, we recommend using this website with a parent or guardian's knowledge and permission. Parents or guardians who have concerns about third-party data practices are encouraged to review the privacy policies linked in this notice.

9. Your rights (GDPR / CCPA)

Because the web designer does not collect or store any personal data, most data subject rights (access, deletion, portability) must be exercised directly with the third-party services that do hold connection metadata:

  • Google (Firebase / Analytics / gstatic.com) — Google Account Privacy Controls
  • GitHub (GitHub Pages) — GitHub Privacy Request
  • jsDelivr — jsDelivr Privacy Policy
  • SheetJS — SheetJS Privacy Policy

California residents (CCPA) and EU/UK residents (GDPR) may have additional rights under applicable law, including the right to know what data is collected, the right to opt out of sale of personal data (no data is sold by the web designer), and the right to non-discrimination.

By clicking "I Agree & Continue" you confirm that you have read and understood this notice and agree to the data practices described above. If you are under 13, please ensure a parent or guardian has reviewed and agreed on your behalf. If you do not agree, please close this page.
