Data & Privacy Notice

What is NOT collected or uploaded

  • Your uploaded videos and images — processed locally in your browser only, never uploaded.
  • Your pose / motion / landmark data — computed locally, never transmitted.
  • Your exported Excel files — generated locally and saved to your device; nothing is sent to any server.

For full details, expand the sections below:

1. Who may log your IP address
  • Web designer — NO
  • Google Firebase Analytics — NO (anonymised)
  • Google Firebase CDN / Database — YES (server logs)
  • GitHub Pages — YES (server logs)
  • jsDelivr CDN — YES (server logs)
  • SheetJS CDN — YES (server logs)

IP logging by these third-party services occurs as a standard part of web hosting and CDN operations and is outside the web designer's control.

2. What data is collected and by whom
  • Visit counter — a single anonymised count is incremented in a Firebase Realtime Database operated by the web designer each time a new browser session begins. No IP address, name, or personal identifier is stored.
  • Firebase Analytics (Google LLC) — may collect approximate geographic region, device type, browser type, OS, language, and session duration. Exact IP is not stored by the web designer; Google applies IP anonymisation where supported. See Google's Privacy Policy.
  • Firebase SDK — Google CDN (gstatic.com) — Firebase library files are fetched from Google's CDN on page load. Google's servers may log your exact IP address, browser User-Agent, referrer URL, and timestamp.
  • Firebase Realtime Database connection — a live HTTPS/WebSocket connection is opened to Google Firebase servers to update the visit counter. Firebase servers may log your exact IP address and connection metadata.
  • GitHub Pages (website host) — GitHub's servers may record your exact IP address, browser User-Agent, referrer, and timestamp for every page request. See GitHub's Privacy Statement.
3. Third-party libraries — CDN metadata

These libraries run entirely in your browser. However, each file is downloaded from an external CDN on first load, exposing your exact IP address, browser type, OS, and referrer to that CDN:

  • MediaPipe Pose (Apache 2.0 License, Google LLC) — loaded from jsDelivr CDN (cdn.jsdelivr.net). All pose detection runs locally via WebAssembly; no frames, images, or landmarks are ever transmitted. See jsDelivr's Privacy Policy.
  • SheetJS Community Edition — loaded from SheetJS CDN (cdn.sheetjs.com). All Excel generation runs locally; no spreadsheet data is transmitted. See SheetJS's Privacy Policy.
4. Cookies and browser storage
  • Google Analytics cookies — Firebase Analytics may set cookies (_ga, _gid) to distinguish sessions. These are governed by Google's cookie policy and can be blocked or deleted via your browser settings.
  • sessionStorage (local only) — two small flags are stored locally: one for consent, one to avoid double-counting visits. This data is never transmitted and is deleted when you close the tab.
  • No tracking cookies by the web designer — no cross-site tracking or browser fingerprinting is performed by the web designer.
5. Data retention
  • Visit counter — anonymous aggregate number, retained indefinitely with no personal data.
  • Google Firebase Analytics — up to 26 months by default, per Google's policy.
  • CDN access logs (gstatic.com, jsDelivr, SheetJS CDN) — per each provider's own policy.
  • GitHub Pages server logs — per GitHub's privacy statement.
  • sessionStorage — deleted automatically when you close the browser tab.
6. Security
  • All connections use HTTPS encryption.
  • No user-generated content (videos, images, pose data) is stored on any server.
  • The web designer does not have access to server logs held by GitHub, Google, jsDelivr, or SheetJS.
7. Children's privacy

This website is suitable for all ages. The web designer does not collect any personal information from any user. Third-party services (Google Firebase, GitHub Pages, CDN providers) may log connection metadata for all visitors regardless of age. If you are under 13, we recommend using this website with a parent or guardian's knowledge and permission.

8. Your rights (GDPR / CCPA)

The web designer holds no personal data. Data subject rights (access, deletion, portability) must be exercised directly with the relevant third parties:

  • Google — Google Account Privacy Controls
  • GitHub — GitHub Privacy Request
  • jsDelivr — jsDelivr Privacy Policy
  • SheetJS — SheetJS Privacy Policy

California (CCPA) and EU/UK (GDPR) residents may have additional rights. No personal data is sold by the web designer.

By clicking "I Agree & Continue" you confirm that you have read and understood this notice and agree to the data practices described above. If you are under 13, please ensure a parent or guardian has reviewed and agreed on your behalf. If you do not agree, please close this page.
